### 🌟 Features
- **Real-Time Sync**: Powered by Firebase Realtime Database for instant tutor-student interaction.
- **Algebra Engine**: Algorithmic problem generation for infinite practice.
- **Gamification**: Success-streak tracking with visual CSS "Glow" effects and high-score persistence.
- **Modern UI**: A professional "Slate & Cyan" engineering aesthetic.

### 🛠 Tech Stack
- **Frontend**: Vite, JavaScript (ES6+), CSS3
- **Backend**: Firebase Realtime Database
- **Animations**: Canvas-Confetti, CSS Keyframes
- **Deployment**: Firebase Hosting + GitHub Actions (CI/CD)

### 📱 Mobile Optimization (iPhone)
SolveSync is currently optimized for desktop and tablet interactions. For the best experience on **iPhone**:
1. Rotate your device to **Landscape Mode**.
2. Set your browser zoom to **50%** (via the `AA` menu in Safari or Browser Settings in Chrome).

### ⚠️ Known Limitations
*   **Multiplication Breakdown**: The "Step-by-Step" problem illustration currently fails to show the specific calculation steps for multiplication problems.
*   **Solution Clarity**: The step-by-step logic currently lacks a simplified, easier-to-understand explanation for complex problem-solving paths. Improvements to pedagogical clarity are ongoing.
