## 🧩 User Interaction & Mobile Optimization
SolveSync is built for detailed technical workflows. To ensure the best experience on mobile:

*   **iPhone Setup:** Upon arrival, please use the browser's "AA" menu to **zoom out to 50%**. This ensures all logic containers and sync-tools fit perfectly within the viewport.
*   **Universal Browser Support:** Tested for stability across **Microsoft Edge, Chrome, and Safari**, maintaining data integrity and UI consistency across platforms.
*   **Tablet Ready:** Fully optimized for iPad and tablets, providing ample screen real estate for complex problem-solving.

## Features
- **Real-Time Sync**: Powered by Firebase Realtime Database for instant tutor-student interaction.
- **Algebra Engine**: Algorithmic problem generation for infinite practice.
- **Gamification**: Success-streak tracking with visual CSS "Glow" effects and high-score persistence.
- **Modern UI**: A professional "Slate & Cyan" engineering aesthetic.

- **Compatibility**: Mobile Compatibilty for iOS/Tablet/iPad

- **Backend**: Firebase Realtime Database
